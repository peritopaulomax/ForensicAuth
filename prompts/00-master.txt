# 00-master.txt - Prompt Master de Contexto

## Contexto do Projeto

Voce esta trabalhando no ForensicAuth: uma plataforma forense digital unificada para peritos criminais da Policia Federal. Consolida tecnicas legadas de analise de imagem, audio, video e PDF em uma aplicacao web profissional.

## Stack

- Backend: FastAPI + Python 3.11 + SQLAlchemy + Celery + Redis
- Frontend: React 18 + TypeScript
- Banco: PostgreSQL 15
- Deploy: Docker + Docker Compose
- Testes: pytest (backend), Vitest (frontend)

## Regras Maximas

1. Toda execucao deve ser pensada do INICIO ate a ENTREGA. Priorize automatismo.
2. Se Yolo Mode estiver ativo, execute ferramentas (Shell, WriteFile, etc.) SEM pedir autorizacao ao usuario.
3. NUNCA entregue trabalho incompleto.
4. TODO codigo gerado DEVE passar nos testes correspondentes antes de avancar.
5. ANTES de escrever qualquer codigo, leia as especificacoes em docs/specs/.
6. Quando o usuario disser "execute prompt X", leia o arquivo prompts/XX-nome.txt e siga EXATAMENTE.
7. Se o contexto esgotar, releia docs/specs/ e AGENTS.md.
8. Bibliotecas forenses especificas dos legados sao INTOCAVEIS. Adaptadores apenas orquestram.
9. Cadeia de custodia digital: todo processamento deve ter hash SHA-256 e ser reprodutivel.
